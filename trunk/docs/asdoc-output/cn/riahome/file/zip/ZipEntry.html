Packagecn.riahome.file.zip
Classpublic class ZipEntry
InheritanceZipEntry Object

一个 ZipEntry 对象指向 zip 文件包里的一个文件（包括文件夹，文件夹是一种特殊的文件）. 在 zip 格式说明书（http://www.pkware.com/documents/casestudies/APPNOTE.TXT）里用 "entry" 一词表示 zip 文件包里的单个文件。 ZipEntry 对象不表示文件实际的字节数据。要取得 zip 文件包里某个文件的字节数据，请查看 ZipFile.getEntryData() 。

注意：ZipEntry 类所有属性均为 internal 类型，即包内可见。你不应该直接访问这些属性，而是通过对应的方法来取得相关属性值。
