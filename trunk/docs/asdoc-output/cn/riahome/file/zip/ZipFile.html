Packagecn.riahome.file.zip
Classpublic class ZipFile
InheritanceZipFile Object

一个 ZipFile 对象代表一个 zip 文件包. 利用 ZipFile 类你可以在flash里创建 zip 文件包，也可以修改已有 zip 文件包的各种信息，还可以添加和删除 zip 文件包里的文件。
