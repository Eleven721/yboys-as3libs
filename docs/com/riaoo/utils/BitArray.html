public function getBitAt(index:uint = 0):Boolean

返回由参数 index 指定位置处的位的值。1 为 true，0 为 false 。

Parameters

index:uint (default = 0) — 一个整数，指定位在 ByteArray 中的位置。第一个位由 0 指示，最后一个位由 ByteArray.length 8 - 1 指示。

Returns
Boolean — 指定索引处的位的值。或者，如果指定的索引不在该 ByteArray 的索引范围内，则返回 false 。

public function setBitAt(index:uint, value:Boolean):void

设置由参数 index 指定位置处的位的值。如果 index 指定位置处的长度大于当前长度，该字节数组的长度将设置为最大值，右侧将用零填充。

Parameters

index:uint — 一个整数，指定位在 ByteArray 中的位置。第一个位由 0 指示，最后一个位由 ByteArray.length 8 - 1 指示。
 
value:Boolean — 要设置的值。true 为 1 ，false 为 0 。
